MetaBLOB



MetaBLOB is an online visualization tool for the metabolism of Physarum polycephalum . It displays a graph that shows a network of known reactions of Physarum and metabolic pathways related to this organism.

With this tool, you can :

  • Upload a JSON file to obtain a visualization of your choice.
  • Modify the layout options to personalize the network.
  • Change the layout algorithm.
  • Show or hide the secondary compounds of the different reactions.
  • Save your work as a PNG picture.
  • Or as different types of files:
    • The JSON file used for the native graph of the tool
    • The XML file in level 2 version 4 we created for the Physarum polycephalum
